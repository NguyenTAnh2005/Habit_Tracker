cấu hình thư mục front 
src/
├── api/            # Chứa file cấu hình Axios (axiosClient.js)
├── assets/         # Ảnh, font
├── components/     # Các nút bấm, input, sidebar dùng chung
├── contexts/       # Quản lý đăng nhập (AuthContext)
├── layouts/        # Bố cục trang (VD: Layout có Sidebar, Layout trống cho Login)
├── pages/          # Các màn hình chính (LoginPage, Dashboard, HabitPage)
├── router.jsx      # Cấu hình đường dẫn
├── App.jsx         # File gốc
└── main.jsx        # File render

Tự động điền http://localhost:8000 vào đầu mỗi link.
